# MERN Stack Practice

Full-stack application built while learning the MERN stack.

## Tech Stack
- **Frontend:** React, React Router, Context API
- **Backend:** Node.js, Express.js
- **Database:** MongoDB + Mongoose
- **Auth:** JWT + bcrypt

## Projects
- `my-backend/` — REST API with authentication
- `my-react-app/` — React frontend with protected routes

## Setup

### Backend
cd my-backend
npm install
npm run dev

### Frontend
cd my-react-app
npm install
npm run dev
